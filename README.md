# Rodrigo Pichara Gomes – Personal Site

A fast, fully static portfolio built with React, Vite, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deployment (GitHub Pages)

1. Push to `main`.
2. GitHub Actions builds the site and deploys the `dist` folder to Pages.

## Notes

- Update the Formspree endpoint in `src/App.tsx` with your live form ID.
- All visual assets live in `src/assets/` and `public/`.
